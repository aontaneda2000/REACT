Elances Externos

Enlaces a páginas de otros sitios web.

  • Ir a google
  • MDN

Atributos recomendados:

  • target="_blank" para abrir el enlace en una nueva pestaña. El valor por defecto es _self
  • rel para indicar la relación entre el documento actual y el documento al que se enlaza.
    • nofollow puedes considerarlo como un valor que se utiliza cuando quieres enlazar a otra página pero sin "avalarlo".
    • noopener evita que la página abierta pueda acceder a la página que la abrió usando JavaScript y asi evitar ataques de phishing.
    • noreferrer evita que la página abierta pueda acceder a la página que la abrió y tampooco dara información sobre quien hizo referencia a ese enlace.
