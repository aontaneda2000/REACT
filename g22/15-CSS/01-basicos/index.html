CSS Básicos

Anatomía de una regla CSS

Una regla CSS está compuesta por un selector y un bloque de declaración que contiene una o más declaraciones. 

        
          selector {
            background-color: dark;
          }

Partes de una regla

  • Selector: Especifica el elemento HTML al que se le aplicará la regla.
    • Elemento: El elemento o los elementos seleccionados por el selector se denominan sujeto del selector.
  • Bloque de declaración: Es el conjunto de declaraciones que se encierran entre llaves.
  • Declaración: Es el conjunto de propiedades y valores que se separan por dos puntos.
  • Propiedad: Son identificadores que indican qué características de estilo deseas cambiar.
  • Valor: Indica cómo quieres que cambien esas características de estilo.
  • Punto y coma: Es el separador de declaraciones.

Agregar CSS a un documento HTML

Existen tres formas de agregar CSS a un documento HTML:

  • En línea
  • Interno
  • Externo

En línea

Se agrega directamente en el elemento HTML que se desea estilizar.

Esta forma de agregar CSS es la menos recomendable, ya que no permite reutilizar el código CSS.

  • Este texto es de color orange
  • Este texto es de color greenyellow
  • Este texto es de color cyan

Interno

Se agrega dentro de la etiqueta <style> del documento HTML.

Esta forma de agregar CSS es recomendable, ya que permite reutilizar el código CSS.

  • Este texto es de color orange
  • Este texto es de color greenyellow
  • Este texto es de color cyan

Externo

Se agrega en un archivo CSS externo y se enlaza al documento HTML.

Esta forma de agregar CSS es la más recomendable, ya que permite reutilizar el código CSS y mantenerlo organizado.

Importancia de la especificidad y el orden

La especificidad y el orden son dos conceptos que debes tener en cuenta a la hora de aplicar estilos a un documento HTML.

Cambiar el comportamiento predeterminado de los elementos

Los navegadores tienen un comportamiento predeterminado para cada elemento HTML. Por ejemplo, los enlaces tienen un subrayado predeterminado, los botones tienen un estilo predeterminado, etc.

Enlaces

Los enlaces tienen un subrayado predeterminado.

Para quitar el subrayado de un enlace, se debe agregar la propiedad text-decoration con el valor none.
